This package contains WPA Portlet Edition (ie Spring Portlet MVC) controller classes for interpolating dynamic tokens in text. The interpolation tokens supported in this package are the same ones supported by the portlet {@link com.hp.it.spf.xa.interpolate.portlet.FileInterpolator} (see).

FileInterpolatorController

The {@link com.hp.it.spf.xa.interpolate.portlet.web.FileInterpolatorController} is an abstract instance of Spring's {@link org.springframework.web.portlet.mvc.AbstractController}, which implements file interpolation during the render phase. Basically it wraps-around {@link com.hp.it.spf.xa.interpolate.portlet.FileInterpolator} to give a Spring controller interface for it. Please see the class documentation, and the SPF Portlet Utilities Developer's Guide, for more information.
