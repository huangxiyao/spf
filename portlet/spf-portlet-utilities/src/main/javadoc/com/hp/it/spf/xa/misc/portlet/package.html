This package contains miscellaneous utility classes for Shared Portal Framework portlet applications.

Consts

The {@link com.hp.it.spf.xa.misc.portlet.Consts} class defines many global attributes potentially useful for SPF-based portlet applications. Generally they are all static and final. For example, the keynames for the user data properties are defined in this class - those are the keynames you use with the {@link com.hp.it.spf.xa.misc.portlet.Utils#getUserProperty(PortletRequest,String)} method. Please see the class documentation for more information.

Utils

The {@link com.hp.it.spf.xa.misc.portlet.Utils} class defines general-purpose utility methods potentially useful for SPF-basd portlet applications. Generally they are all static methods. For example, using this class you can get data about the user (eg, whether or not the user is logged-in; and properties/attributes of the user) and about the portal request (eg, the portal site name and URL). Please see the class documentation for more information.
