The projects in this directoty contain the OpenPortal files that were modified by HP team and which are not
yet part of the official OpenPortal code base. We most likely work with that team to include these 
modifications in their source repository.

The "version.txt" file contains information about the portlet-container and wsrp OpenPortal SVN revision
numbers that correspond to the files in this directory. If we integrate with newer versions of these 
projects, the "version.txt" file must be updated appropriately.
