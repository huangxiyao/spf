This package contains miscellaneous utility classes for both portlets and portal components in the Shared Portal Framework.

Consts

The {@link com.hp.it.spf.xa.misc.Consts} class defines many global attributes potentially useful for SPF-based portal components and portlet applications. Generally they are all static and final. For example, the keynames for the user data properties are defined in this class. The portal {@link com.hp.it.spf.xa.misc.portal.Consts} and portlet {@link com.hp.it.spf.xa.misc.portlet.Consts} classes inherit from this one.

CountryRegionMapping

The {@link com.hp.it.spf.xa.misc.CountryRegionMapping} class implements a mapping between ISO 3166-1 country codes and the HP business region: Americas, EMEA, or Asia-Pacific-Japan. Please see the class documentation for more information.

Environment

The {@link com.hp.it.spf.xa.misc.Environment} class is a singleton providing current domain and server information about the WebLogic managed environment. Please see the class documentation for more information.

Utils

The {@link com.hp.it.spf.xa.misc.Utils} class defines general-purpose utility methods potentially useful for SPF-based portal components and portlet applications. Generally they are all static methods. The portal {@link com.hp.it.spf.xa.misc.portal.Utils} and portlet {@link com.hp.it.spf.xa.misc.portlet.Utils} classes inherit from this one. Please see the class documentation for more information.
