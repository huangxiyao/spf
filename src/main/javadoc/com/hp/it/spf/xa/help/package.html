This package contains abstract base classes for plain Java objects related to online help in both the portal and portlet frameworks of the Shared Portal Framework.

Note: SPF portal and portlet developers should never need to use the Java classes in this package directly. Generally they should import the classes provided for them in the portal and portlet frameworks:

  • Portal developers should see the documentation in {@link com.hp.it.spf.xa.help.portal}
  • Portlet developers should see the documentation in {@link com.hp.it.spf.xa.help.portlet}

However, developers who want to extend these base classes to produce new, concrete kinds of online help, may do so.

HelpProvider

The {@link com.hp.it.spf.xa.help.HelpProvider} class is the abstract base class for all kinds of online-help provider, like {@link com.hp.it.spf.xa.help.portlet.ClassicContextualHelpProvider} for portlets, and {@link com.hp.it.spf.xa.help.portal.ClassicContextualHelpProvider} and {@link com.hp.it.spf.xa.help.portal.ClassicGlobalHelpProvider for portal components. (Help providers are plain Java objects which know how to generate the HTML markup for particular kinds of online help. They are used by online-help JSP tags, and also by the internationalization code for injecting online-help markup into messages.)

ContextualHelpProvider

The {@link com.hp.it.spf.xa.help.ContextualHelpProvider} class is the abstract base class for all contextual help providers, like {@link com.hp.it.spf.xa.help.portlet.ClassicContextualHelpProvider} for portlets, and {@link com.hp.it.spf.xa.help.portal.ClassicContextualHelpProvider} for portal components. (What is a help provider? See HelpProvider discussion above.)

ClassicContextualHelpProvider

The {@link com.hp.it.spf.xa.help.tag.ClassicContextualHelpProvider} class is the abstract base class for classic contextual help providers, like {@link com.hp.it.spf.xa.help.portlet.ClassicContextualHelpProvider} for portlets, and {@link com.hp.it.spf.xa.help.portal.ClassicContextualHelpProvider} for portal components. (What is a help provider? See HelpProvider discussion above.)

HelpUtility

The {@link com.hp.it.spf.xa.help.HelpUtility} class is the abstract base class for all kinds of online-help utilities, like {@link com.hp.it.spf.xa.help.ContextualHelpUtility} for contextual help, and {@link com.hp.it.spf.xa.help.portal.GlobalHelpUtility} for global help (only available to portal components). This class provides common utility methods for parsing messages (for help injection), for example.

ContextualHelpUtility

The {@link com.hp.it.spf.xa.help.ContextualHelpUtility} class provides utility methods for all kinds of contextual help. This class provides a utility method for parsing messages (for contextual-help injection), for example.
