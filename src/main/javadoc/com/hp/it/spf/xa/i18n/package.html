This package contains plain Java classes for internationalizing both portlet applications and portal components in the Shared Portal Framework.

Note: SPF portal and portlet developers should never need to use the Java classes in this package directly. Import the portal or portlet subclasses instead - eg, {@link com.hp.it.spf.xa.i18n.portlet.I18nUtility} for portlets, and {@link com.hp.it.spf.xa.i18n.portal.I18nUtility} for portal components.

I18nUtility

The {@link com.hp.it.spf.xa.i18n.I18nUtility} class provides methods for:

  • getting localized displayable values for various objects (timezones, user names, etc)
  • converting between different representations of locale ({@link java.util.Locale}, ISO 639-1, RFC 3066, etc)
  • sorting locales
  • and more.
Please see the class documentation for more information.
