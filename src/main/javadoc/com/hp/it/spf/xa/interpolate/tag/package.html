This package contains abstract base classes for JSP tags related to file interpolation, in both the portal and portlet frameworks of the Shared Portal Framework.

Note: SPF portal and portlet developers should never need to use the Java classes in this package directly. Use the respective JSP tags instead which extend and implement these classes for portal components and portlets:

  • Portal developers should see the tag documentation in {@link com.hp.it.spf.xa.interpolate.portal.tag}
  • Portlet developers should see the tag documentation in {@link com.hp.it.spf.xa.interpolate.portlet.tag}

However, developers who want to extend these base classes to produce new kinds of interpolator tags, may do so.

FileInterpolatorBaseTag

The {@link com.hp.it.spf.xa.interpolate.tag.FileInterpolatorBaseTag} class is the abstract base class for all file interpolation tags, like <spf-file-portlet:interpolate> for portlets and <spf-file-portal:interpolate> for portal components.
