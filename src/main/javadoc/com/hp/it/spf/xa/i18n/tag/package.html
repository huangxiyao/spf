This package contains abstract base classes for JSP tags related to internationalization in both the portal and portlet frameworks of the Shared Portal Framework.

Note: SPF portal and portlet developers should never need to use the Java classes in this package directly. Use the respective JSP tags instead which extend and implement these classes for portal components and portlets:

  • Portal developers should see the tag documentation in {@link com.hp.it.spf.xa.i18n.portal.tag}
  • Portlet developers should see the tag documentation in {@link com.hp.it.spf.xa.i18n.portlet.tag}

However, developers who want to extend these base classes to produce new kinds of internationalization tags, may do so.

ParamBaseTag

The {@link com.hp.it.spf.xa.i18n.tag.ParamBaseTag} class is the abstract base class for string parameter tags (for message tags), like <spf-i18n-portlet:param> for portlets and <spf-i18n-portal:i18nParam> for portal components.

MessageBaseTag

The {@link com.hp.it.spf.xa.i18n.tag.MessageBaseTag} class is the abstract base class for message tags, like <spf-i18n-portlet:message> for portlets and <spf-i18n-portal:i18nValue> for portal components.

LocalizedFileURLBaseTag

The {@link com.hp.it.spf.xa.i18n.tag.LocalizedFileURLBaseTag} class is the abstract base class for tags which express a localized file-download URL, like <spf-i18n-portlet:localizedFileURL> for portlets and <spf-i18n-portal:localizedFileURL> for portal components.

ContextualHelpParamBaseTag

The {@link com.hp.it.spf.xa.i18n.tag.ContextualHelpParamBaseTag} class is the abstract base class for all contextual help parameter tags (for message tags), like <spf-i18n-portlet:classicContextualHelpParam> for portlets and <spf-i18n-portal:i18nClassicContextualHelpParam> for portal components.

ClassicContextualHelpParamBaseTag

The {@link com.hp.it.spf.xa.i18n.tag.ClassicContextualHelpParamBaseTag} class is the abstract base class for classic contextual help parameter tags (for message tags), like <spf-i18n-portlet:classicContextualHelpParam> for portlets and <spf-i18n-portal:i18nClassicContextualHelpParam> for portal components.
