This package contains abstract base classes for text interpolation, for both portlet applications and portal components in the Shared Portal Framework.

Note: SPF portal and portlet developers should never need to use the Java classes in this package directly. Import the portal or portlet subclasses instead - eg, {@link com.hp.it.spf.xa.interpolate.portlet.FileInterpolator} for portlets, and {@link com.hp.it.spf.xa.interpolate.portal.FileInterpolator} for portal components.

FileInterpolator

The {@link com.hp.it.spf.xa.interpolate.FileInterpolator} class is an abstract base class for text interpolation from files, using the TokenParser below. The file may be localized; the class looks for the best-fit localized version of the file when loading it. Please see the class documentation for more information.

TokenParser

The {@link com.hp.it.spf.xa.interpolate.TokenParser} class is an abstract base class for text interpolation from strings. The parsing of common portal/portlet tokens like {LANGUAGE-CODE}, {SITE-URL:spec}, etc, is done here. Please see the class documentation for more information.
