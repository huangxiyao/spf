This package contains abstract base classes for JSP tags related to online help in both the portal and portlet frameworks of the Shared Portal Framework.

Note: SPF portal and portlet developers should never need to use the Java classes in this package directly. Use the respective JSP tags instead which extend and implement these classes for portal components and portlets:

  • Portal developers should see the tag documentation in {@link com.hp.it.spf.xa.help.portal.tag}
  • Portlet developers should see the tag documentation in {@link com.hp.it.spf.xa.help.portlet.tag}

However, developers who want to extend these base classes to produce new kinds of online help tags, may do so.

HelpBaseTag

The {@link com.hp.it.spf.xa.help.tag.HelpBaseTag} class is the abstract base class for all online-help tags, like <spf-help-portlet:classicContextualHelp> for portlets, and <spf-help-portal:classicContextualHelp> and <spf-help-portal:classicGlobalHelp> for portal components.

ContextualHelpBaseTag

The {@link com.hp.it.spf.xa.help.tag.ContextualHelpBaseTag} class is the abstract base class for all contextual help tags, like <spf-help-portlet:classicContextualHelp> for portlets and <spf-help-portal:classicContextualHelp> for portal components.

ClassicContextualHelpBaseTag

The {@link com.hp.it.spf.xa.help.tag.ClassicContextualHelpBaseTag} class is the abstract base class for classic contextual help tags, like <spf-help-portlet:classicContextualHelp> for portlets and <spf-help-portal:classicContextualHelp> for portal components.
