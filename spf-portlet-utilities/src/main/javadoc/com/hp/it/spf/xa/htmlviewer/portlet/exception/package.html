This package contains SPF portlet exception classes for the html-viewer portlet. They are included here in the SPF portlet utilities, instead of inside the html-viewer project itself, so that they are available in the SPF portlet utilities JAR for developers to extend as needed.

InputErrorException

The {@link com.hp.it.spf.xa.htmlviewer.portlet.exception.InputErrorException} class is the exception class for user input errors (ie administrator input errors during config mode).

InternalErrorException

The {@link com.hp.it.spf.xa.htmlviewer.portlet.exception.InternalErrorException} class is the exception class for system errors (eg user attempts view mode before any view file has been configured).
